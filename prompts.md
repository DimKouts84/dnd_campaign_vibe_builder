# Prompt Templates
Use these templates to generate high-quality, structured D&D content. Copy-paste these into your chat with the agent and fill in the brackets `[...]`.

## 1. Campaign Brainstorming
> **Context:** Start of a new campaign or arc.
> **Goal:** Generate unique hooks and settings.

```markdown
**Task:** Brainstorm 3 distinct D&D campaign concepts.
**Theme:** [e.g., Eldritch Horror, High Seas Swashbuckling, Political Intrigue]
**Tone:** [e.g., Gritty, Whimsical, Epic]
**Starting Level:** [1-20]
**Constraints:**
- Must include a major faction called [Faction Name].
- Ideally set in [Region/Setting].

**Output Format:**
For each concept, provide:
1. **Title**
2. **The "Hook"** (One paragraph pitch)
3. **The Big Bad** (Who is the antagonist?)
4. **Key Conflict**
```

## 2. NPC Generator
> **Context:** need a quick NPC for a shop or quest giver.
> **Goal:** Complete statblock and personality.

```markdown
**Task:** Create a detailed NPC.
**Name:** [Optional Name]
**Role:** [e.g., Shopkeeper, Guard Captain, Cultist Leader]
**Race/Class:** [e.g., Tiefling Rogue]
**Vibe:** [e.g., Nervous, Arrogant, Helpful but shady]

**Output detailed markdown:**
- **Name & Description:** Visuals + Voice acting cues.
- **Personality:** Ideal/Bond/Flaw.
- **Secret/Motivation:** What do they want?
- **Statblock:** Suggested Statblock (use SRD reference like "Use *Spy* stats but add *Misty Step*").
```

---

## 3. Encounter Builder (Combat)
> **Context:** Planning a battle.
> **Goal:** Balanced and tactical combat.

```markdown
**Task:** Design a combat encounter.
**Party:** [Number] players at Level [Level].
**Difficulty:** [Easy / Medium / Hard / Deadly].
**Environment:** [e.g., Crumbling Bridge, Swamp with fog].
**Enemy Archetype:** [e.g., Undead, Bandits, Fey].

**Requirements:**
1. **Enemy Roster:** List monsters and counts. Check Action Economy.
2. **Tactics:** Round-by-round strategy for the monsters.
3. **Terrain Hazards:** Add 1 interactive element (e.g., explosive barrels, slippery moss).
4. **Loot:** Gold + 1 interesting item.
```

---

## 4. Quest Structure
> **Context:** Flesh out a plot hook.
> **Goal:** A multi-step quest with varied gameplay.

```markdown
**Task:** Outline a side-quest.
**Hook:** [e.g., Missing child, Cursed sword found].
**Steps:**
1. **Investigation:** Clues to find.
2. **Exploration/Social:** Travel or negotiation.
3. **Climax:** Combat or High-Stakes Skill Challenge.
**Reward:** XP and Gold/Item.
```

---

## 5. Balance Check (Nerf/Buff)
> **Context:** You have a cool homebrew monster but worry it's broken.
> **Goal:** Adjust stats for fairness.

```markdown
**Task:** Analyze this monster for balance.
**Target Party:** [Number] players, Level [Level].
**Monster Stats:** [Paste Stats or Describe Ability].

**Analysis Needed:**
- Is the *Damage per Round* too high?
- Is the *AC/HP* appropriate?
- Are there "Save or Die" mechanics?
**Recommendation:** Suggest specific number tweaks to make it fair but challenging.
```
